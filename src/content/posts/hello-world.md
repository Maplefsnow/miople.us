这个博客用 [Astro](https://astro.build) 搭建，部署在 GitHub Pages 上，外观沿用了我们小窝的粉色玻璃质感 ✿

- 中英双语都可以发
- 文章可以加标签
- 支持代码高亮和 MDX

```js

miople.forEach((name) => console.log(`hello, ${name}!`));
```
